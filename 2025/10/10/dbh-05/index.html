Inner JOIN

If we want to see all 2018 American movies in the database, we need to resort to an extended kind of JOIN, known as the outer JOIN (the regular JOIN is also known as the inner JOIN). An outer JOIN build the result set with NULLs if we can’t find a match in the outer joined table.

There are three kinds of outer JOINs:

  • Left outer JOIN
  • Right outer JOIN
  • Full outer JOIN The mostly commonly used one is the left outer JOIN. A right outer JOIN can always be expressed in a left one, and the full outer JOINs are rarely used.

The NULL Problem:

Arithmetic operators:

  • col+NULL=NULL
  • col-NULL=NULL
  • col*NULL=NULL
  • col/NULL=NULL

Relational operators:

  • col>NULL=NULL
  • col<NULL=NULL
  • col>=NULL=NULL
  • col<=NULL=NULL
  • col!=NULL=NULL
  • col is NULL=TRUE or FALSE

Logical operators:

  • TRUE and NULL=NULL
  • FALSE and NULL=FALSE
  • TRUE or NULL=TRUE
  • FALSE or NULL=NULL
  • If col is 'a' or 'b', then the result is TRUE.
  • Otherwise the result is NULL.
  • If col is 'a' or 'b', then the result is FALSE.
  • Otherwise the result is NULL.
